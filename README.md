# FTXUI-Tetris-Assignment
这是一个关于《FTXUI 俄罗斯方块》大作业的精炼总结：  本项目采用 C++ 开发，深度集成 **FTXUI** 库构建终端 GUI。针对 Windows 平台，通过 `CMakeLists.txt` 的精准过滤与 UNICODE 宏定义，解决了头文件路径依赖与编码乱码难题，实现了稳定跨平台构建。  游戏核心利用 `std::thread` 与矩阵碰撞算法，实现了自动下落、消行计分与旋转逻辑。在 AI 辅助下，重构了命名空间冲突，优化了渲染比例，将原本畸形的地图转化为具备彩色方块与网格背景的街机风格界面，显著提升了视觉体验。本项目通过 GitHub 进行版本管理，完美符合课程对 AI 实践与规范化工程的要求。
